###**RStudio快捷键** Ctrl + L 清除控制台输出 Ctrl + Enter 运行光标所在行的R代码 或 当前选中行的R代码 Ctrl + Shift + S 运行当前脚本文件 Ctrl + D 删除整行 --- ###**命令行安装flash** ``` sudo apt-get update sudo apt-get install flashplugin-installer ``` ###**命令行安装forefox** ``` sudo apt-get update sudo apt-get install firefox ``` ###**命令行安装vim** ``` sudo apt-get update sudo apt-get install vim ``` 配置vim编辑器 终端切到在主目录下（cd ~） 执行命令 vim .vimrc打开或创建vim的配置文件 加入自己的配置方案 例如： `set nu` "是显示行号 `set showmatch` "显示括号匹配等 ###**jd-gui 反编译java源码** 下载`jd-gui-0.3.3.linux.i686.tar.gz` 执行`tar -xzvf jd-gui-0.3.3.linux.i686.tar.gz` ``` sudo mv jd-gui /usr/local/bin ``` 就可以运行了，只需将.class文件拖拽到jd-gui中 ###**MySQL安装** ``` sudo apt-get install mysql-server ``` 登录：`mysql -uroot -p` ####**常用操作** ``` //想myuser使用mypassword从任何主机连接到mysql服务器 mysql> grant all privileges on *.* to myuser@'%' identified by mypassword with grant option; //创建用户 mysql> create user 'hadoop' identified by 'hadoop'; //授权 mysql> grant all on *.* to hadoop@'%' with grant option; mysql> quit; [root@hadoop9 hadoop]# mysql -uhadoop -p mysql> create database hive; mysql> quit; ```
